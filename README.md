# Trilha2_Projeto2
 uma página web que contenha um input para titulo e outro para escrever um conteúdo. Vamos simular a criação de um post semelhante ao facebook, linkedIn... Onde daremos um título, e escreveremos um texto qualquer e postamos. E o mais legal deste projeto, é que estaremos fazendo uma comunicação com uma API aberta. Simulando um pouco do mundo real.

# COOMO TESTAR
Abra index.html diretamente no navegador ou use a opção Open with Live Server no VS Code para ter recarregamento automático. Digite um título e um conteúdo com pelo menos 10 caracteres, clique em “Publicar agora” e confira o título/texto retornados na seção “Prévia do post publicado”.
